# Research Data Registry for Process Science

This data overview was created as part of the [NFDIxCS](https://nfdixcs.org/) project. It provides an overview of existing public data available for research in process science. The overview can be accessed here: https://promilab.github.io/ResearchDataRegistry/ .

Currently the collection contains data from a variety of domains such as: 
- Software Engineering
- Human-Computer Interaction
- Logistics
- IoT
- Business Administration
- Healthcare
- Gaming
- Robotics
- Blockchain Technology

## Data Sources

The data is retrieved from the following sources:
- BPI Challenges: https://tfpm.compute.dtu.dk/competitions-awards/bpi-challenge
- Process Discovery Contest Collection: https://www.tf-pm.org/competitions-awards/discovery-contest
- Blockchain Data for Process Mining: https://ingo-weber.github.io/dapp-data/
- IoT Data for Process Mining: https://zenodo.org/communities/iopt/records?q=&l=list&p=1&s=10&sort=newest
- OCEL Event Logs Overview: https://www.ocel-standard.org/event-logs/overview/
- Event Logs (XES, OCEL, CSV): https://www.processmining.org/event-data.html
- Event Logs (XES, CSV): https://zenodo.org/records/16268743

Some of these sources have been described and documented in the following papers:
- Bandara, H. D., Bockrath, H., Hobeck, R., Klinkmüller, C., Pufahl, L., Rebesky, M., van der Aalst, Will, & Weber, I. (2021, September). Event logs of ethereum-based applications. In BPM’21: international conference on business process management, Rome, Italy.
-	Pohl, T., Berti, A., Qafari, M. S., & van der Aalst, W. M. (2023). A collection of simulated event logs for fairness assessment in process mining. arXiv preprint arXiv:2306.11453.
-	Costa, A., Eroglu, S. Y., Andree, K., & Pufahl, L. (2025). A Collection of Publicly Available Event Logs Enhanced by Metadata.

## Metadata Model
The description of the data in the overview follows the metadata model ProMeta as described in the following paper:
- Schreiber, C., Fritsch, A., Schiefer, G., & Oberweis, A. (2025). Towards a Metadata Model for Process Data. In EMISA.
